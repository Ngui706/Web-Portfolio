This project is my responsive personal portfolio website. It showcases basic front-end web development skills using HTML and CSS, with interactive hover effects to display skill proficiency levels.

Features

Responsive Navigation Bar

Fixed top navigation with anchor links

Mobile-friendly behavior (menu hides on small screens)

About Me Section

Brief professional summary

Centered content for readability

Skills Section

Skill cards with hover interaction

Percentage proficiency displayed on hover

Demonstrates CSS positioning, transitions, and pseudo-interactions

Projects Section

Grid-based project layout

Image thumbnails with descriptions

Responsive grid using CSS Grid

Contact Section

Simple contact details

Dark-themed section for visual separation

Modern UI Design

Google Fonts (Poppins)

Neutral color palette

Clean spacing and typography

Technologies Used

HTML5

Semantic layout

Section-based structure

Accessibility-friendly markup

CSS3

Flexbox and Grid layouts

Hover effects and transitions

Responsive design with media queries

Google Fonts

Poppins font family for modern typography
